A fully responsive, animated, cloud/DevOps-themed portfolio built with HTML, CSS, and JavaScript.

## Deploy
- GitHub Pages: push to `main` and enable Pages for the repo root.
- Netlify/Vercel: import the repo and deploy as a static site.

## Configure
- Replace `assets/Anirudh_Trivedi_Resume.pdf` with your resume file.
- Update LinkedIn/GitHub/Blog links in `index.html`.
- EmailJS: in `index.html` and `js/main.js`, replace `PUBLIC_KEY_PLACEHOLDER`, `SERVICE_ID`, and `TEMPLATE_ID`.
- Replace sound placeholders in `assets/sounds/`.

## Dev
Open `index.html` in a browser. No build step required.
